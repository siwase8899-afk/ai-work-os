# AI Work OS

회사 업무와 개인 업무를 분리하고, 반복 업무를 AI 기반 대시보드와 리포트로 관리하기 위한 웹 기반 발표 자료입니다.

## Pages

- `index.html`: 발표용 메인 슬라이드
- `stock-dashboard.html`: 주식 종목 관리 대시보드
- `wellness-coach.html`: AI Wellness Coach 초안

## Deploy

GitHub Pages에서 `GitHub Actions` 배포 소스를 선택하면 `.github/workflows/deploy-pages.yml` 워크플로우가 정적 파일을 배포합니다.
